## About

This is the source code for Derrick Mbesa's personal portfolio website. Built as a single-page HTML/CSS/JS site and hosted on GitHub Pages, it showcases services, results, credentials, and case studies for healthcare providers looking for remote administrative support.

---

## Features

- Responsive single-page layout
- Sticky navigation with smooth scrolling
- Services grid with 6 core offerings
- Metrics and results showcase
- Credentials and certifications section
- Tools & platforms proficiency list
- Client testimonial block
- Case studies grid
- Calendly booking integration
- HIPAA compliance indicators

## Local Development

No build tools or dependencies needed. Just open the file:

```bash
# Clone the repository
git clone https://github.com/Mbesad123/derrick-mbesa-portfolio.git

# Navigate into the folder
cd derrick-mbesa-portfolio

# Open in your browser
open index.html
# Or on Windows:
start index.html
```

---

## Deployment

This site is deployed automatically via **GitHub Pages** on every push to the `main` branch.

To deploy updates:

```bash
git add .
git commit -m "Update portfolio content"
git push origin main
```

The live site updates within 1–2 minutes at:
`https://mbesad123.github.io/derrick-mbesa-portfolio/`

## Customization

All content lives in `index.html`. Key sections to update:

| What | Where in the file |
|------|------------------|
| LinkedIn URL | Footer — `Connect` column |
| WhatsApp number | Footer — replace `https://wa.me/` with `https://wa.me/YOUR_NUMBER` |
| Calendly link | Already set to `calendly.com/mbesad6` throughout |
| Email | Already set to `mbesad6@gmail.com` throughout |
| Case study links | Services grid — each "Read case study" anchor |
